# The Java Calculator with Graphical User Interface

## How to use?

- Compile the project
  ```shell
  $ make compile
  ```

- Run the program
  ```shell
  $ make run
  ```

- Clean the files
  ```shell
  $ make clean
  ```

## Notes
- 如果需要執行模組程式可以參照以下方法：
  - `dirName` 為模組名稱(資料夾)
  - `fileName` 為包含 `main` Method的檔案名稱
    ```shell
    $ java dirName.fileName
    ```
- 安裝 [Java JDK](https://www.oracle.com/java/technologies/downloads/#jdk18-windows)
